Обов'язково:

Зверстати шопінг ліст. 

В рішенні обов’язково повинні бути використані селектори:

- за елементом
- за класом
- за атрибутом
- селектор сусідів “+”
- :hover
- :active
- :checked
- :before і :after
- :nth-child

Для плавних анімацій використай CSS-код:

- *{transition: all .1s ease-in;}
